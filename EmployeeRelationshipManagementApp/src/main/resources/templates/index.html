Employee Relationship Management Application

Application Overview

The Employee Relationship Management (ERM) application is a comprehensive solution developed to streamline and enhance the management of employee information within an organization. This application is built using:

  • Java: Core programming language used for the application logic.
  • Spring Boot: Framework for creating stand-alone, production-grade Spring-based applications.
  • MySQL: Relational database management system for storing employee data.
  • Thymeleaf: Template engine for rendering dynamic web content.
Key Features
  • Employee Management: Add, edit, delete, and view employee details.
  • Department Management: Manage different departments within the organization.
  • User Authentication: Secure login and access control for different user roles.
  • User-friendly interface for managing employee records.
  • Validation of critical fields such as email and mobile number to ensure data integrity.
